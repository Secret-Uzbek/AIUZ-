# AIUZ

AI Ethics-based Universal Learning Zone.

This repository is a live practical node inside the wider Terra / FMP ecosystem.
It belongs to the practice trunk, not to the whole archive, not to the full theory trunk,
and not to the standalone legal trunk.

## Role in the ecosystem

`AIUZ-` is for practical, public-facing, implementation-oriented material connected to:

- ethical educational technology;
- multilingual interfaces and learning logic;
- AIUZ platform architecture and documentation;
- public repository-level artifacts meant to be readable by humans first.

This repository is **not** the total storage of Terra.
It must not be used as a blind sink for all files, backups, session residues, or shadow exports.
